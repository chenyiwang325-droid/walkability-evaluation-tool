# 愉悦性层级图像

将需要评价愉悦性的街道断面图像放置在此文件夹中。

## 愉悦性说明

关注街道空间的审美体验和活动活力，提升步行过程中的愉悦感受。

包括建筑风貌、景观美学、多样性和活力等维度。

## 细分维度

- **建筑一致性与尺度不适宜 (PR1)**：街道空间尺度的适宜性与建筑风貌的协调性
- **美学吸引力不足 (PR2)**：街道环境的视觉美感，直接体现为街道界面的设计美感
- **多样性与复杂性不足 (PR3)**：街道特色的丰富与独特程度，体现街道在构成要素上的多元性
- **活力与活动水平不足 (PR4)**：街道的活动水平与空间活力，主要体现为行人的数量与活动状态
