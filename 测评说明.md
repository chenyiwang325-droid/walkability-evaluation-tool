# 街道安全性专家测评说明

## 一、测评简介

本测评针对街道空间的安全性,组织专家对街景图像进行评价,产出专家评价结果。测评分两批独立进行:

| 批次 | 评价者 | 图像 | 评价方式 |
|---|---|---|---|
| 外部盲评 | 编号 1–13(13人) | 现有 52 张街景 | 仅依据图像与安全性定义盲评 |
| 内部扩展 | 编号 14–26(13人) | 新增 52 张街景 | 可参考要素识别、知识库与归因项 |

每批 52 张图像,每位评价者分配 20 张(循环分配),每张图像由恰好 5 位评价者覆盖,共 26 位评价者、两批共 104 张图像。

## 二、评价链接

**优先使用(国内访问更快、稳定):**

```
https://walkability-evaluation-tool.pages.dev/
```

**备用(GitHub Pages,国内偶有加载慢):**

```
https://chenyiwang325-droid.github.io/walkability-evaluation-tool/eval/
```

评价者打开链接,输入分配给自己的编号即可进入对应评价页面(编号决定其看到的图像与评价内容,评价者无需了解分组)。

### 网络适配性说明

- **优先使用 Cloudflare Pages 链接**:部署于全球 CDN,国内访问速度更快、带宽无限,适合多名评价者同时访问,长期免费有效。
- **GitHub Pages 链接作为备用**:内容与主链接完全一致、同步更新。若主链接在个别网络环境下打不开,可换用此备用链接,或更换网络、刷新重试。
- **纯前端、无需安装**:工具为静态网页,在任意现代浏览器(Chrome / Edge / Safari / Firefox)中打开即可,不依赖服务器,断网也不丢失已填数据。
- **数据本地存储**:评分自动保存在评价者浏览器的本地存储中,中途切换网络、关闭浏览器后再次打开同一链接(同一浏览器)可续评;完成后导出文件交回即可。
- **建议**:评价者使用稳定的 Wi-Fi 或 4G/5G 网络;若图片加载较慢,可等待或刷新单张。

## 三、评价工具简介

纯前端网页工具,无需安装,在浏览器中打开即可使用。

**评价流程**
1. 输入评价者编号进入;
2. 左侧显示街景图像,右侧为打分面板;
3. 完成一张后点"下一张"继续(支持键盘 ←/→);
4. 全部 20 张完成后点"导出",保存结果文件交回。

**打分面板内容**
- 外部盲评:安全性定义 + 1–5 诊断评级 + 问题归因(填写归因名称与归因分析,需结合街景图像中的空间要素,可添加多条;每条归因下可嵌套填写优化策略(策略方向+理由)与实施举措(具体做法+实施说明);或勾选"无明显问题",可附说明,二者互斥)。
- 内部扩展:知识库(定义/归因含义/评级标准)+ 要素识别结果 + 总体诊断评级 + SR1 自然监视 / SR2 环境失序 分项诊断评级 + 问题归因复选 + 按"归因→策略→举措"链路勾选优化策略与实施举措。

**辅助功能**
- 首次进入自动弹出分步导览(高亮框出各功能并编号说明),之后可点顶部"指引"按钮重看;
- 评分自动保存到浏览器,可随时退出续评;
- 顶部"进度详情"下拉查看完成情况并跳转任意一张;
- 顶部视图切换:"逐张评价"逐张填写;"画册视图"总览全部图像已填的诊断评级、问题归因、优化策略与实施举措标签,便于完成后对照检查,点击卡片可跳回该图;内部扩展另有"历史参照",可查看本人在前两轮评价中的选择记录(已按现行口径整理,仅供对齐参考);
- 一张图所有回答填齐才算"完成",导出时提示完成进度。

## 四、结果导出

- 格式:Excel(.xlsx)或 JSON,文件名含评价者编号,如 `安全性评价_评价者03.xlsx`;
- 字段:评价者编号、图像 pid、总体评级、问题归因等(外部/内部两套字段格式分开,由编号区分);
- 评价者在本地导出后,将文件交回汇总即可。
