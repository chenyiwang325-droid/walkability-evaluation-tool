# 街道可步行性专家评价工具 - 使用说明

## 一、启动工具

1. 解压缩文件夹
2. 双击 `启动工具.bat` 文件
3. 等待浏览器自动打开（如未自动打开，手动访问 http://localhost:3000）

> 如果无法启动，请确保已安装 Node.js（下载地址：https://nodejs.org）

## 二、选择评价层级

从顶部导航栏选择要评价的层级：
- **通达性**：街道是否可供步行，是否具备人行通过的条件
- **安全性**：步行道本身的社会空间安全性
- **舒适性**：步行时的方便程度和舒适程度
- **愉悦性**：街道空间的审美体验和活动活力

## 三、加载图像

选择层级后，点击**「一键加载」**按钮，系统会自动加载该层级的待评价图像。

## 四、选择评价模式

加载图像后，选择评价模式：

| 模式 | 说明 |
|------|------|
| **盲评模式**（推荐） | 直接进入打分，无法查看模型输出，避免评价偏差 |
| **浏览模式** | 可以先浏览图像和模型输出，再进入打分 |

选择后点击**「进入评价」**。

## 五、进行评价

在打分界面：

1. **查看图像**：左侧显示待评价的街道断面图像
2. **查看参考数据**：图像下方显示模型输出的参考信息
3. **进行评分**：右侧面板选择评分（1-5分）
4. **选择问题归因**（如有）：勾选存在的问题类型
5. **自动保存**：评分会自动保存

使用底部导航按钮可切换上一张/下一张图像。

## 六、导出结果

评价完成后：

1. 点击右上角**「导出」**按钮
2. 选择导出格式（推荐 Excel 格式）
3. 保存文件

## 七、评价其他层级

1. 点击左上角**「返回」**按钮回到欢迎页
2. 从顶部导航栏选择新的层级
3. 重复步骤三至六

## 评分标准

| 分值 | 含义 |
|------|------|
| 1 | 很差 |
| 2 | 较差 |
| 3 | 一般 |
| 4 | 较好 |
| 5 | 很好 |

详细评分标准请参考 `docs/可步行性评价评分标准.md` 文件。

## 常见问题

**Q: 数据会丢失吗？**
A: 数据自动保存在本地，关闭浏览器后再次打开数据仍然存在。建议评价完成后及时导出备份。

**Q: 如何继续之前的评价？**
A: 重新启动工具后，之前的数据会自动加载，可以从上次的位置继续评价。

**Q: 多人如何独立评价？**
A: 每位评价者应独立运行本工具（复制整个文件夹到各自电脑），评价完成后各自导出结果文件汇总即可。
