# 街道可步行性专家评价工具

基于建成环境步行性评估框架的专业评价系统，支持多维度街道空间评价。

## 功能特性

- **四层级独立评价**：通达性、安全性、舒适性、愉悦性
- **多维度细分评价**：每个层级包含多个细分维度
- **问题归因选择**：支持多选问题归因
- **数据导出**：支持 Excel 和 JSON 格式导出
- **自动保存**：评分自动保存

## 快速开始

### 环境要求

- Node.js 14+（可选，用于服务器模式）

### 使用方式

#### 方式一：直接打开（推荐新手）

直接双击 `index.html` 文件在浏览器中打开即可使用。

> 注意：此方式数据保存在浏览器本地存储中，清除浏览器数据会丢失评价记录。建议评价完成后及时导出。

#### 方式二：服务器模式（推荐）

1. 确保已安装 [Node.js](https://nodejs.org/)
2. 双击 `启动工具.bat` 文件
3. 服务器将自动启动并打开浏览器
4. 访问 http://localhost:3000

> 服务器模式支持数据持久化到本地文件。

## 图像文件放置

将待评价的街道断面图像按层级放置在 `images` 文件夹下：

```
images/
├── 通达性/          ← 通达性层级图像
├── 安全性/          ← 安全性层级图像
├── 舒适性/          ← 舒适性层级图像
└── 愉悦性/          ← 愉悦性层级图像
```

## 评价层级说明

| 层级 | 说明 |
|------|------|
| 通达性 | 关注街道是否可供步行，是否具备人行通过的条件 |
| 安全性 | 关注步行道本身的社会空间安全性（不含交通安全） |
| 舒适性 | 关注步行时的方便程度和舒适程度 |
| 愉悦性 | 关注街道空间的审美体验和活动活力 |

## 评分标准

采用5级量表：

| 分值 | 含义 |
|------|------|
| 1 | 很差 |
| 2 | 较差 |
| 3 | 一般 |
| 4 | 较好 |
| 5 | 很好 |

详细的评分标准请参考 `docs/可步行性评价评分标准.md`

## 数据导出

评价完成后，点击「导出」按钮可选择导出格式：

- **Excel 格式**：便于后续数据分析
- **JSON 格式**：保留完整数据结构

## 多人使用说明

如果多位专家需要使用本工具：

1. 每位专家应独立运行本工具
2. 数据存储在各自的浏览器本地存储中，互不影响
3. 评价完成后，各自导出结果文件
4. 将导出的文件汇总即可
