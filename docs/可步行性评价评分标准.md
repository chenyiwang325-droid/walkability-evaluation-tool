# 街道可步行性专家评价标准

## 一、总体评价等级定义

| 评价 | 定义                      |
| -- | ----------------------- |
| 很差 | 该维度存在严重问题，对可步行性产生重大负面影响 |
| 较差 | 该维度存在明显问题，对可步行性产生较大负面影响 |
| 一般 | 该维度表现一般，对可步行性影响中性       |
| 较好 | 该维度表现较好，对可步行性有一定正面影响    |
| 很好 | 该维度表现优秀，对可步行性产生显著正面影响   |

***

## 二、各层级评价标准

### 1. 通达性

**层级定义**：关注街道是否可供步行，是否具备人行通过的条件。具体体现为两类情况：一是不存在可供行走的人行通道，人行路径被完全堵死、不可逾越（如施工完全阻挡人行道等，不包括等级较低的人行混行道路）；二是街道宽度过宽，且缺乏天桥、下穿道或二次过街路口等行人过街设施，导致行人过街困难。

| 评价     | 判断标准                            |
| ------ | ------------------------------- |
| **很差** | 可见的步行道路被完全阻断，无法继续前进             |
| **较差** | 两侧没有独立的可步行空间（如受施工影响导致步行道路被完全占用） |
| **一般** | 街道宽度过宽，且缺乏过街设施，导致行人过街困难         |
| **较好** | 没有独立的步行空间（混行道路），或仅单侧存在独立步行空间    |
| **很好** | 两侧均存在通达的步行空间                    |

#### 街道连通性不足 (AR1)

- **很差**：可见的步行道路被完全阻断，无法继续前进
- **很好**：两侧均存在通达的步行空间

***

### 2. 安全性

**层级定义**：安全性代表了对于街道空间对人身体伤害、心理压力和其他潜在威胁的感知，该层级主要关注的是步行道本身或是其社会空间的安全性，不包括交通安全。

| 评价     | 判断标准                                      |
| ------ | ----------------------------------------- |
| **很差** | 自然监视严重缺失（完全封闭、无照明、无活动），环境严重失序（破败、垃圾、危险施工） |
| **较差** | 自然监视明显不足（视线受阻、照明不足），环境存在明显失序（破损、杂乱）       |
| **一般** | 自然监视存在一定不足（部分区域视线受阻），环境存在轻微失序（局部杂乱）       |
| **较好** | 自然监视基本充足（视觉通透性良好），环境整体整洁（仅有轻微杂乱）          |
| **很好** | 自然监视充分（高透明度、充足照明、商铺活动），环境完全整洁有序           |

#### 自然监视不足 (SR1)

- **很差**：步行空间被封闭界面完全包围，无路灯照明，无任何商铺、开放空间或日常活动
- **很好**：高底层透明度形成良好视觉通透性，路灯照明充足，商铺、开放空间、日常活动等形成持续监视

#### 环境失序 (SR2)

- **很差**：存在多处破旧建筑、严重损坏的设施、大量垃圾堆积、废弃空间，施工行为缺乏防护
- **很好**：街道环境完全整洁有序，无任何破旧建筑、损坏设施、杂乱电线、垃圾或不当施工

***

### 3. 舒适性

**层级定义**：舒适性关注人步行时的方便程度，是否有充足的空间、平整的地面进行步行活动，以及能够减弱步行疲劳度或抵御不良天气的设计。

**⚠️ 舒适性评价采用降级辅助判断**：
一般基于较舒适的标准考虑：天气缓冲设施遮蔽充分、人行道宽敞平整、机动交通隔离满足需求（已经存在隔离或没有强制隔离需求），为很好标准。存在某些问题时，开始降级。

| 评价     | 判断标准                                    |
| ------ | --------------------------------------- |
| **很差** | 机动交通隔离严重不足，步行空间被完全占用或阻断，无任何遮蔽或休憩设施      |
| **较差** | 机动交通隔离明显不足，步行空间狭窄或被严重占用，缺乏遮蔽和休憩设施       |
| **一般** | 机动交通隔离存在一定不足，步行空间基本可用但有局部问题，遮蔽设施有限      |
| **较好** | 机动交通隔离基本有效，步行空间充足且维护良好，有一定的遮蔽和服务设施      |
| **很好** | 机动交通隔离充分有效，步行空间宽敞平整无占用，遮蔽设施连续，休憩和服务设施完善 |

#### 机动交通隔离不足 (CR1)

- **很差**：街道缺少必要的隔离设施（护栏、隔离带等明确设施及行道树、路缘石等隐性隔离）
- **很好**：设有完善的隔离设施，或设有交通稳静设施
- **注意**：当道路较宽、交通性较强时对机动交通隔离要求更加严格。

#### 步行空间条件较差 (CR2)

- **很差**：人行道被部分占用（车辆/局部施工），人行道路面严重破损
- **很好**：人行道宽敞、平整，无任何占用，铺装质量优良

#### 街道家具设施不足 (CR3)

- **很差**：无任何遮蔽设施，无任何休憩或服务设施
- **很好**：遮蔽设施充足且连续，休憩设施充足，服务设施完善
- 注意：一般以行道树等天气缓冲设施为首要判断，休憩服务设施为提升性判断

***

### 4. 愉悦性

**层级定义**：愉悦性关注街道空间的审美体验和活动活力，提升步行过程中的愉悦感受，包括建筑风貌、景观美学、多样性和活力等维度。

**⚠️ 愉悦性评价采用加减辅助判断**：

由于愉悦性的整体判断不太容易达成一致的标准，采用以下加减逻辑：

| 评价         | 判断标准                      |
| ---------- | ------------------------- |
| **很好** | 各个维度都相对比较好                |
| **较好** | 有一定出众的亮点（基于几个维度考虑）        |
| **一般** | 平平无奇的街道，非常常见，没有什么特色（默认基准） |
| **较差** | 能够明显感受到某个维度表现较差           |
| **很差** | 整体愉悦性表现非常差                 |

#### 建筑一致性与尺度不适宜 (PR1)

- **很差**：建筑风格完全混乱，建筑高度与周边严重不协调，街道高宽比极度失衡，街道界面破碎无序
- **很好**：建筑风格高度统一且与周边环境协调，街道高宽比舒适宜人，行道树种植与建筑形成良好协同，街道界面连贯有序

#### 美学吸引力不足 (PR2)

- **很差**：植被搭配单一、形态色彩及季相变化匮乏，沿街建筑缺乏吸引力，无公共艺术装置，无美学照明设计，街道视觉单调乏味
- **很好**：植被搭配丰富且美观，建筑立面设计精美，有公共艺术装置和景观照明设计，街道视觉形象出色

#### 多样性与复杂性不足 (PR3)

- **很差**：完全缺乏历史建筑或特色元素，建筑形态完全统一，景观样式单一，色彩搭配单调，沿街功能极其单一，街道完全缺乏辨识度
- **很好**：拥有历史建筑或丰富的地方特色文化元素，建筑形态多样，景观样式丰富，沿街功能多元，街道具有独特的辨识度
- **注意**：
  - 对于**以历史文化风貌为主的街道**：关注历史建筑与传统风貌建筑、地方特色文化元素等相关要素
  - 对于**一般街道**：关注独特建筑、色彩元素、特色景观元素等视觉类要素，以及沿街功能多样性

#### 活力与活动水平不足 (PR4)

- **很差**：无开放空间，建筑未形成连续沿街界面，首层可进入性业态不足，沿街业态门店密度低，人流活跃度极低
- **很好**：有丰富的开放空间和活动场所，建筑形成积极连续的沿街界面，首层业态多样且可进入性高，沿街门店密集，人流活跃

## 三、评价判断要点

### 通达性

- 核心关注：街道是否可供步行，是否具备人行通过的条件
- 关键判断：步行路径是否存在、是否被阻断、街道宽度是否过宽

### 安全性

- 核心关注：步行空间本身的安全感（不包括交通安全）
- 自然监视：视觉通透度、照明、商铺活动、日常活动等
- 环境失序：破旧建筑、杂乱街道、损坏现象、废弃空间、垃圾、施工行为等

### 舒适性

- 核心关注：步行时的方便程度
- 机动交通隔离：**仅当道路较宽、交通性较强时才有需求**，较窄的街道不强制要求，存在路缘石、行道树等隐性隔离即可。
- 步行空间条件：宽度、占用、维护
- 街道家具设施：遮蔽设施、休憩设施、服务设施

### 愉悦性

- 核心关注：审美体验和活动活力
- **评价方式**：加减制（默认一般，有亮点升至较好，各维度都好升至很好；明显有维度差降至较差，有维度很差降至很差）
- 建筑一致性与尺度：风格统一性、高度协调性、空间尺度
- 美学吸引力：植被景观、建筑美感、公共艺术、照明设计
- 多样性与复杂性：**对于以历史文化风貌为主的街道关注历史建筑、特色文化元素；对于一般街道关注独特建筑、色彩景观等视觉要素**
- 活力与活动水平：开放空间、业态密度、人流活跃度
